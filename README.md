# Esteban Aguilar - Portfolio

## About

Academic portfolio showcasing my journey in Computer Science Engineering at Hillsborough Community College.

## Technologies

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Development

To run this project locally:

```sh
# Clone the repository
git clone <YOUR_GIT_URL>

# Navigate to the project directory
cd <YOUR_PROJECT_NAME>

# Install dependencies
npm i

# Start the development server
npm run dev
```

## Deployment

This project can be deployed to any static hosting service that supports React applications.
